# Urban Crime Rate Analysis

## Project Overview
This project analyzes urban crime data to uncover patterns, trends, and insights.  
The focus is on **data collection, cleaning, exploratory analysis, and visualization** of crime rates in different urban areas.

## Key Steps
1. **Data Collection**  
   - Gathered crime datasets from official sources and public records.

2. **Data Cleaning**  
   - Handled missing values, duplicates, and inconsistent entries.  
   - Ensured all columns are properly formatted for analysis.

3. **Exploratory Data Analysis (EDA)**  
   - Generated summary statistics for numeric and categorical features.  
   - Counted occurrences in categorical columns to understand distributions.

4. **Visualization**  
   - Created plots to visualize trends, distributions, and hotspots.  
   - Charts include histograms, barplots, and scatter plots to reveal patterns.
